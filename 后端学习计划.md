#### 1. CommonResult 统一返回

**核心文件：**

**核心结构：**
```java
{
    "code": 0,           // 0=成功, 非0=失败
    "msg": "成功",        // 提示信息
    "data": {...}        // 实际数据
}
```

**关键方法：**
- `CommonResult.success(data)` - 成功返回
- `CommonResult.error(code, msg)` - 失败返回

#### 2. 错误码体系

**全局错误码：**
- 文件：[GlobalErrorCodeConstants.java](file:///Users/zkr/工作/学习/项目/ruoyi-vue-pro/yudao-framework/yudao-common/src/main/java/cn/iocoder/yudao/framework/common/exception/enums/GlobalErrorCodeConstants.java)

**错误码范围：**
- `0` - 成功
- `400-499` - 客户端错误（参数错误、未登录、无权限）
- `500-599` - 服务器错误
- `900-999` - 自定义错误

**业务错误码：**
- 文件：[ErrorCodeConstants.java](file:///Users/zkr/工作/学习/项目/ruoyi-vue-pro/yudao-module-system/src/main/java/cn/iocoder/yudao/module/system/enums/ErrorCodeConstants.java)
- 格式：`模块号_子模块号_序号`，例如 `1_002_000_000`

**实战：**

```java
// 返回: {"code": 1002000000, "msg": "登录失败，账号密码不正确"}
```

**学习内容：**

#### 1. Service 接口

**示例文件：**
- 接口：[AdminUserService.java](file:///Users/zkr/工作/学习/项目/ruoyi-vue-pro/yudao-module-system/src/main/java/cn/iocoder/yudao/module/system/service/user/AdminUserService.java)

**接口定义：**
```java
public interface AdminUserService {
    AdminUserDO getUserByUsername(String username);
    Long createUser(UserSaveReqVO reqVO);
    void updateUser(UserSaveReqVO reqVO);
    void deleteUser(Long id);
}
```

#### 2. Service 实现

**示例文件：**
- `yudao-module-system/src/main/java/cn/iocoder/yudao/module/system/service/user/AdminUserServiceImpl.java`

**关键注解：**
```java
@Service                    // Spring Bean
@Slf4j                     // 日志
@Transactional             // 事务管理
@Validated                 // 参数校验
```

**典型方法结构：**
```java
@Override
@Transactional(rollbackFor = Exception.class)
public Long createUser(UserSaveReqVO reqVO) {
    // 1. 参数校验
    validateUser(reqVO);
    
    // 2. VO 转 DO
    AdminUserDO user = BeanUtils.toBean(reqVO, AdminUserDO.class);
    
    // 3. 密码加密
    user.setPassword(encryptPassword(reqVO.getPassword()));
    
    // 4. 插入数据库
    userMapper.insert(user);
    
    // 5. 返回结果
    return user.getId();
}
```

#### 3. 对象转换（BeanUtils）

**使用方式：**
```java
// VO 转 DO
AdminUserDO user = BeanUtils.toBean(reqVO, AdminUserDO.class);

// DO 转 VO
UserRespVO vo = BeanUtils.toBean(user, UserRespVO.class);

// 集合转换
List<UserRespVO> list = BeanUtils.toBean(userList, UserRespVO.class);
```

#### 4. 事务管理

**何时使用：**
```java
@Transactional(rollbackFor = Exception.class)
public void someMethod() {
    // 涉及多个表的操作
    userMapper.insert(user);
    roleMapper.insert(role);
    permissionMapper.insert(permission);
}
```

**完成标准：**
✅ 理解 Service 接口与实现分离的原因  
✅ 能够编写带事务的业务方法  
✅ 掌握 BeanUtils 对象转换

### Day 6: VO 对象体系

**学习目标：**
- 理解 VO 对象的分类和用途
- 掌握分页查询的标准写法

**VO 分类：**

1. **ReqVO（请求对象）**
   - 用于接收前端传来的参数
   - 包含校验注解（@NotBlank、@Email 等）

2. **RespVO（响应对象）**
   - 用于返回给前端的数据
   - 字段可能与 DO 不同

3. **PageReqVO（分页请求）**
   - 继承 PageParam
   - 包含 pageNo、pageSize 字段

4. **ExcelVO（导入导出）**
   - 用于 Excel 导入导出
   - 包含 @ExcelProperty 注解

**示例：**
```java
// 分页请求
public class DictTypePageReqVO extends PageParam {
    @ApiModelProperty("类型名称")
    private String name;
    
    @ApiModelProperty("类型编码")
    private String type;
}

// 响应对象
public class DictTypeRespVO {
    @ApiModelProperty("编号")
    private Long id;
    
    @ApiModelProperty("类型名称")
    private String name;
}
```

**流程图：**
```
1. 用户请求 POST /system/auth/login
   ↓
2. AuthController.login()
   ↓
3. AdminAuthService.login()
   ├─ 验证图形验证码
   ├─ 验证用户名密码
   │  ├─ 查询用户
   │  ├─ 校验密码（BCrypt）
```

#### 2. 权限注解

**常用注解：**
```java
@PreAuthorize("@ss.hasPermission('system:user:query')")    // 权限校验
@PreAuthorize("@ss.hasRole('admin')")                      // 角色校验
@PermitAll                                                  // 允许匿名访问
@DataPermission(enable = false)                            // 禁用数据权限
```

#### 3. JWT Token

**Token 结构：**
- Header（头部）
- Payload（载荷）：包含用户ID、用户类型等
- Signature（签名）

**Token 使用：**
```java
// 生成 Token
String token = JwtUtils.createToken(userId, userType);

// 解析 Token
Long userId = JwtUtils.getUserId(token);

// 验证 Token
JwtUtils.verifyToken(token);
```

#### 2. 缓存注解

**常用注解：**
```java

@CacheEvict(value = "user", key = "#id")          // 删除缓存
```

#### 3. Redisson 分布式锁

**使用场景：**
- 防止重复提交
- 定时任务防并发
- 库存扣减

```java
@Resource
private RedissonClient redissonClient;

RLock lock = redissonClient.getLock("lock:key");
try {
    lock.lock();
    // 业务逻辑
} finally {
    lock.unlock();
}
```
