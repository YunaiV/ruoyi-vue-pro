# YMQ · 羽毛球助手

> 让一群人一起打球这件事，从拉群到散场，全程顺滑。

一款服务业余羽毛球社群的微信小程序，专注解决「组织一场球」全链路痛点：智能对阵表生成、发现附近活动、比赛实时记分、段位评测、个性化训练计划。

## 五大核心功能

> 小程序底部 5 个 Tab：`🏸 活动` · `🏆 比赛` · `📊 段位` · `💪 训练` · `📚 教学`

### 1. 活动（创建 + 接龙 + 对阵表 + 发现附近）⭐ 核心爆点
召集人创建活动 → 群友扫码接龙 → 一键生成均衡对阵表 → 一键转发回群。
缺人时可勾选"对外公开"→ 活动出现在附近球友的发现页 → 申请加入需召集人审核 + 信用分护航。
**对应代码**：`ymq-module-activity` + `ymq-module-social`

### 2. 比赛
- 接力赛 100 分制（男单/女单/男双/女双/混双轮转）
- 淘汰赛（4/8/16/32 人单淘汰）
- 裁判端 + 观众端实时同步
**对应代码**：`ymq-module-match`

### 3. 段位评测 + AI 教练彩蛋
自创 13 级段位体系（D1-S），5 分钟答题测出水平。
评测报告页内嵌"AI 教练问答"彩蛋（看握拍/站位/装备），接 Claude/GPT 多模态 API。
**对应代码**：`ymq-module-rating`

### 4. 训练成长 ⭐ 留存核心
基于段位评测自动生成 4 周个性化提升计划。技能树覆盖基础/进攻/防守/战术 4 大类约 30 个技能。
打卡 → 累计积分 → 4 周后复测看进步 → 分享对比海报。
**对应代码**：`ymq-module-training`

### 5. 教学知识库（小程序 Tab + H5 站点）
精选教程内容，**双形态**：小程序"教学"Tab 给已登录用户随时翻看；H5 站点对外做 SEO 长尾流量。
是段位评测的"推荐课程"和训练成长的"教学视频"的内容支撑层。
**对应代码**：`ymq-module-content`

## 技术栈

### 前端
- **小程序**：原生 + Vant Weapp UI
- **H5**：VitePress（知识库）
- **运营管理后台**：芋道自带 Vue3 后台

### 后端
- **骨架**：**YunaiV/ruoyi-vue-pro（芋道源码）2.4+**（37.1k★ 国内最热门若依商业增强版）
- **语言**：Java 17（芋道默认；可升 21）
- **框架**：Spring Boot 3.2+
- **ORM**：MyBatis-Plus
- **数据库**：MySQL 8.0
- **缓存**：Redis 7
- **实时通信**：WebSocket
- **微信 SDK**：**binarywang/WxJava 4.8+**（芋道已集成，开箱即用登录/支付/订阅消息/公众号）

### 基础设施
- **服务器**：腾讯云轻量应用服务器（2 核 4G）
- **对象存储**：腾讯云 COS
- **CDN**：腾讯云 CDN
- **错误追踪**：Sentry

### AI 能力（分阶段引入，详见 [1-产品愿景.md §AI 能力路线](docs/1-产品愿景.md#ai-能力路线)）
- **v1**：AI 教练问答彩蛋（接 Claude 4.x / GPT-4o 多模态 API）
- **v1.5**：多轮对话 + 个人档案接入
- **v2**：单动作姿态分析（MediaPipe BlazePose + 规则引擎）
- **v3**：长视频段位评估

## 关键架构决策

| 决策 | 选择 | 理由 |
|---|---|---|
| 后台骨架 | **芋道 ruoyi-vue-pro**（v1.2 切换） | 微信全链路开箱即用，节省 5-7 周；文档最完整；37.1k★ 社区最活跃 |
| 启动方式 | **单启动 + 接口前缀**（v1.2 调整） | 芋道原生支持；MVP 阶段单进程够用；MAU > 5w 再拆 |
| 用户体系 | sys_user 和 t_user 分离 | 运营和 C 端权限完全隔离 |
| 前端 | 微信小程序原生 + Vant Weapp | 性能最优、维护简单 |
| 发现功能 | 只做"开放活动"不做"找人" | 避免社交监管、保护女性用户 |
| 知识库 | H5 而非小程序内 | SEO 长尾流量、可独立做内容站 |
| 训练计划内容 | MVP 100% 整合公开教程（B 站/视频号外链） | 0 内容成本，先验证留存 |
| AI 策略 | v1 只做问答彩蛋，分四阶段引入 | 避免被 AI 视频识别拖垮 MVP；不影响段位评分降低用户投诉风险 |
